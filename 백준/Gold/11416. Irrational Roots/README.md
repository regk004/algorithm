### 문제 설명

<p>Let n be a natural number, n ≤ 8. Consider the following equation:</p>

<p style="text-align:center">x<sup>n</sup> + c<sub>n-1</sub>x<sup>n-1</sup> + c<sub>n-2</sub>x<sup>n-2</sup> + … + c<sub>1</sub>x + c<sub>0</sub> = 0</p>

<p>where c<sub>n-1</sub>, c<sub>n-2</sub>, …, c<sub>1</sub>, c<sub>0</sub> are integers and c<sub>0</sub> ≠ 0.</p>

<p>It is known that all the n roots of the equation are real numbers. We consider that each root r of the equation satisfies the condition: -10 ≤ r ≤ 10. Also, there might be roots that appear more than once.</p>

<p>Find the number of irrational roots of the equation (an irrational root is a root that is an irrational number).</p>

### 입력 

 <p>The input file contains a single test. The first line of the input file contains the value of n. The second line contains the values of c<sub>n-1</sub>, c<sub>n-2</sub>, …, c<sub>1</sub>, c<sub>0</sub>: each two consecutive values are separated by a single space.</p>

### 출력 

 <p>The result will be written to standard output.</p>
